# Data Visualization with Java, MongoDB and R (GraalVM) ## 📌 Project Description This project implements a real-time data visualization system using: - Java (GraalVM) - MongoDB - R (lattice library) The system reads numeric data from MongoDB in cyclic order and sends the values to an R function for plotting. The purpose of this project is to demonstrate: - MongoDB data handling - Java–R interoperability using GraalVM - Real-time data visualization using R --- ## 🧱 System Architecture The system works as follows: 1. A CSV file (100 rows) is manually imported into MongoDB. 2. Only **one selected column** (e.g., Col-1) is used. 3. The Java program: - Reads one consecutive row every second - Loops back to the beginning after row 99 (cyclic reading) - Sends the double value to an R plotting function 4. R plots the values using xyplot() from the lattice library. --- ## 🗂️ Data Source - CSV file contains 100 rows. - Only **one column** is used for this project. - Data must be manually imported into MongoDB before running the program.
